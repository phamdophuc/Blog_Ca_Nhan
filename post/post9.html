Đoạn Code Đầu Tiên Khi Học JavaScript

Khi bắt đầu học JavaScript trên Cisco, mình thực sự ấn tượng với cách ngôn ngữ này tạo ra sự tương tác động. Đoạn code đầu tiên mình yêu thích là tạo một bộ đếm thời gian, giúp mình hiểu về cách JavaScript thao tác với DOM (Document Object Model).

            let count = 0;
            const counterElement = document.getElementById('counter');
            const incrementButton = document.getElementById('increment');
            
            incrementButton.addEventListener('click', () => {
                count++;
                counterElement.textContent = `Count: ${count}`;
            });

Cách hoạt động:

  • document.getElementById truy xuất các phần tử HTML bằng ID của chúng.
  • Một biến count được khởi tạo để lưu trữ giá trị bộ đếm.
  • Gắn sự kiện click cho nút bằng addEventListener. Khi nút được nhấn, giá trị count tăng thêm 1.
  • counterElement.textContent thay đổi nội dung hiển thị trên giao diện, cập nhật giá trị bộ đếm mới.
  • Dựa vào giá trị của biến isPrime, chương trình sẽ in kết quả tương ứng.

Đoạn code này giúp mình hiểu cách xử lý sự kiện và cập nhật giao diện động trong JavaScript. Cisco đã hướng dẫn cách kết nối logic với giao diện một cách mượt mà, làm rõ cách các thành phần tương tác với nhau.
