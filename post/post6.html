Đoạn Code Đặc Biệt: Chương Trình Kiểm Tra Số Nguyên Tố

Một đoạn code khác mà mình rất thích trong quá trình học Java tại Cisco là bài tập kiểm tra số nguyên tố. Đoạn code này không chỉ đơn thuần kiểm tra tính chất toán học mà còn giúp mình hiểu rõ hơn về cách tối ưu hóa thuật toán với vòng lặp và điều kiện. Đây là đoạn code: 

            Scanner scanner = new Scanner(System.in);
            System.out.print("Nhập một số: ");
            int number = scanner.nextInt();
            boolean isPrime = true;
            
            if (number < 2) {
                isPrime = false;
            } else {
                for (int i = 2; i <= Math.sqrt(number); i++) {
                    if (number % i == 0) {
                        isPrime = false;
                        break;
                    }
                }
            }
            
            if (isPrime) {
                System.out.println(number + " là số nguyên tố.");
            } else {
                System.out.println(number + " không phải là số nguyên tố.");
            }

Cách hoạt động:

  • Sử dụng Scanner để nhận đầu vào từ người dùng.
  • Xác định nếu số nhỏ hơn 2 thì không phải số nguyên tố.
  • Sử dụng vòng lặp chạy từ 2 đến căn bậc hai của số nhập vào để kiểm tra.
  • Nếu tìm thấy bất kỳ số nào chia hết cho số nhập vào, vòng lặp dừng ngay lập tức nhờ lệnh break, giúp tiết kiệm thời gian xử lý.
  • Dựa vào giá trị của biến isPrime, chương trình sẽ in kết quả tương ứng.

Bài tập này giúp mình nhận thức được tầm quan trọng của việc tối ưu hóa thuật toán, nhất là khi làm việc với các bài toán có phạm vi lớn.
